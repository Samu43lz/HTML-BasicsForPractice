Etiquetas Semanticas Estructurales

Etiqueta <div>

Es una etiqueta contenedora que no tiene semantica

Etiquetas con Semantica (con significado)

Etiqueta <header>

header - Cabecera de un sitio web o de una sección (generalmente se le coloca el nav aqui)

Etiqueta <main>

main - Define la sección principal del documento, sólo puede existir una etiqueta main por documento

Etiqueta <footer>

footer - Pie de página de un sitio web o de una sección

Etiqueta <nav>

nav - Representa una navegación (un conjunto de enlaces que formen un menú de navegación) tambien puede ser una botonera.

Etiqueta <article>

article - representa una sección de autocontenido (que por si sola se explica).

Etiqueta <aside>

aside - Representa contenido complementario o secundario (como un "articulos relacionados"). Tambien nos sirve para colocar zonas de publicidad

Etiqueta <section>

section - Representa una sección de contenido genérico.

Etiqueta <address>

address - Representa una dirección fisica.

Etiquetas de Bloque VS Etiquetas de Linea

La etiqueta de bloque más representativa por excelencia es <div>, una etiqueta que genera saltos de linea con sus etiquetas hermanas. (osea abarca todo el espacio de la linea donde este)
La etiqueta de linea por excelencia es <span>, una etiqueta de Linea sólo ocupa espacio según su contenido. (lo necesario) Otra <span> de ejemplo
